# أُفق — منصة تعلّم شاملة وفق UDL

منصة عربية متجاوبة تعمل مباشرة في المتصفح، وتضم تسجيل حساب ودخولاً محلياً، لوحة تعلم، مقررات، مشغل درس متعدد الوسائط، اختبارات، مجتمعاً تعليمياً، تقويماً، ملفاً شخصياً، وأدوات وصول متكاملة.

## التشغيل

افتح `index.html` مباشرة، أو شغّل خادماً محلياً:

```powershell
python -m http.server 4173
```

ثم افتح `http://localhost:4173`.

## النشر

المشروع ثابت بالكامل ويمكن رفع مجلد `Force` كما هو إلى Netlify أو Vercel أو GitHub Pages. لا توجد خطوة بناء ولا متغيرات بيئة مطلوبة.

## بيانات العرض

من صفحة الدخول استخدم زر «الدخول بالحساب التجريبي»، أو أنشئ حساباً جديداً. تُحفظ البيانات والتقدم وإعدادات الوصول في `localStorage` داخل المتصفح.
